## Spring Boot Movies Rest API

Spring boot project with the next dependencies / starters:
* Mysql Driver
* Spring Data JPA
* Spring Web
* Lombok
* Spring Test
* Spring validation
* Springdoc openapi webmvc ui

Rest API application with access to mysql database.
